## ✨ Overview

This repository holds the source code for my personal portfolio website a single-page site that introduces me, showcases my skills, and highlights the projects I've built. It's designed around a **dark background with coral (`#f5857c`) accents** and **glassmorphism cards**, styled entirely with hand-written CSS (no frameworks) and finished off with the **Poppins** typeface.

## 🎯 Features

- 🎨 **Dark + coral glassmorphism aesthetic** — soft-blurred cards over a moody dark backdrop
- 📱 **Fully responsive** — adapts cleanly across desktop, tablet, and mobile
- 🍔 **Animated hamburger menu** for mobile navigation
- 🌊 **Scroll-reveal animations** powered by the `IntersectionObserver` API
- ⬆️ **Back-to-top button** that appears on scroll
- 📄 **One-click resume download**
- 🔗 **Live social & project links** (GitHub, LinkedIn, Instagram, Twitter)

## 🚀 Getting Started

Since this is a static site, there's nothing to install or build.

```bash
# 1. Clone the repository
git clone https://github.com/abhinavxjha/myportfolio.git

# 2. Move into the folder
cd myportfolio

# 3. Open it in your browser
open index2.html      # macOS
start index2.html      # Windows
xdg-open index2.html   # Linux
```

Or, use the **Live Server** extension in VS Code for auto-reloading while you edit.
